DESARROLLO DIGITAL

Programación.

Aprende lógica de programación, desarrollo web moderno y creación de aplicaciones digitales.

HTML & CSS

Aprende a crear sitios web modernos, atractivos y funcionales desde cero utilizando las tecnologías fundamentales del desarrollo web. Conocerás la estructura de una página mediante HTML y el diseño visual con CSS, aprendiendo a organizar contenido, agregar imágenes, enlaces, formularios, menús de navegación y elementos interactivos.

Lógica

Desarrolla la capacidad de analizar problemas y crear soluciones de manera estructurada utilizando el pensamiento lógico y computacional. Aprenderás conceptos fundamentales como algoritmos, diagramas de flujo, variables, operadores, estructuras condicionales, ciclos repetitivos y resolución paso a paso de problemas.
